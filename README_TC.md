# Hentaium

Hentaium是一個基於Chromium的瀏覽器

它⸺為了順暢遊玩網頁遊戲而生。

它不再限制頁面在背景執行，
即使你切換到其他分頁或者將瀏覽器最小化，
遊戲也能繼續在背景執行。

所以您無須再為Chrome/Firefox的背景執行限制設定而煩惱！

## 功能

1. 不再限制背景執行
2. Manifest V2擴充功能

**由於Hentaium並不支援Chrome同步功能，所以你需要自行匯入書籤。**

## 版本

Hentaium目前使用Chromium M138 LTS作為基礎版本。

## 編譯

編譯方法與Chromium官方文件所述的一樣，但您需要自行使用 `git am` 或白 `git apply` 以套用相關修改。
